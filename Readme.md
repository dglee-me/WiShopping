# 위쇼핑(WISHopping)
__위쇼핑__은 Wish + Shopping 의 합성어로, 내가 원하는 제품을 살 수 있는 쇼핑몰 프로젝트입니다.

__소셜 커머스__ 방식의 쇼핑몰로 판매를 하고 싶은 회원은 판매자 등록 후 허가를 받아 자유로이 상품을 판매할 수 있으며,<br/>
원하는 상품을 구매하고 싶은 회원은 해당 상품 페이지에서 간단하게 구매할 수 있습니다.

## 프로젝트 진행 동기 및 목적
1. 요즘 시대에 가장 많이 사용되는 쇼핑몰 구조인 소셜 커머스 의 구현 및 꽤 자주 접하게 될 아이템으로 주제 선정
2. Java, Oracle, 웹프로그래밍(HTML,CSS,JavaScript,AJAX 등), Spring Framework 등 실무에 필요한 주요 기술들의 공부 및 포트폴리오 목적
